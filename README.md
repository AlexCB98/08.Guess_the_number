# Guess The Number (Python Game)

A simple interactive number guessing game built in Python as a learning project.

## How the game works

- The computer chooses a random number between **0 and 100**
- You choose a difficulty:
  - **Easy** → 10 lives 
  - **Hard** → 5 lives 
- You try to guess the number
- After each guess, you get feedback:
  - Too high 
  - Too low 
- The game ends when:
  - You guess correctly 
  - You run out of lives 

---

## Features

- Difficulty selection (Easy / Hard)
- Lives system displayed with 
- Replay system (play multiple times)
- ASCII art logo for better UI
- Clear feedback for each guess
